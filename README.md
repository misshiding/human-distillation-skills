## 关于 PR

欢迎提交 PR 收录新的 Skills！

### 提交流程

1. **Fork 本仓库**
2. **找到对应分类**，在表格末尾添加新行
3. **按格式填写**，确保序号正确
4. **提交 PR**，我会尽快审核

### 提交格式

```markdown
| 序号 | [项目名称](链接) | 简短描述 |
```

### 要求

- 必须是真实存在的 Skill 项目（有 GitHub 仓库）
- 描述限 50 字以内，一句话说明核心价值
- 描述用语风格与该分类保持一致
- 同一分类内按提交时间排序

### 不被收录的类型

以下类型的 Skill 不会被收录：

- **未经授权的真实人物** - 蒸馏真人需获得本人授权，特别是私人关系（前任、暗恋对象等）
- **未成年人** - 任何涉及未成年人的蒸馏项目
- **隐私信息密集型** - 需要大量个人隐私数据且无授权证明的
- **有明显商业侵权风险的** - 如假冒名人、仿冒品牌
- **恶意用途** - 目的为欺诈、冒充、操控等

> 本列表收录的是 Skill **工具/方法论本身**，不代表对蒸馏行为合法性的背书。使用者需自行确保不侵犯他人权益。
